✅ БЫСТРОЕ РЕШЕНИЕ (5 минут):

   1. Установите ngrok: https://ngrok.com/download
   
   2. Запустите в новом терминале:
      ngrok http 5000
   
   3. Скопируйте URL (например: https://abc123.ngrok.io)
   
   4. Обновите webhook в Payme:
      https://business.paycom.uz/
      → Касса 670a6af1a048b8a82254e446
      → Webhook URL: https://abc123.ngrok.io/api/payme/webhook
   
   5. Готово! Теперь оплата будет работать! ✅
